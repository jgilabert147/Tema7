Lenguajes de marcas y sistemas de gestión de información

El módulo de Lenguajes de marcas es un módulo de 1º con una carga horaria de 3 horas semanales impartidas a lo largo de 3 trimestres.

Los contenidos del módulo son los siguientes:

  • Características de los lenguajes de marcas
  • Lenguajes para el almacenamiento y transmisión de información
  • El lenguaje XML
  • XPath y XQuery
  • El lenguaje JSON
  • El lenguaje Markdown
  • El lenguaje HTML 5
  • El lenguaje CSS3
  • El lenguaje de programación Javascript
